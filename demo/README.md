# 电子吧唧SDK Demo应用

这是一个演示如何使用电子吧唧SDK的示例应用。

## 功能

1. **设备连接**
   - 扫描蓝牙设备
   - 连接/断开设备
   - 显示连接状态

2. **图片同步**
   - 选择图片文件
   - 自动转换为设备格式（240x240, bin格式）
   - 上传到设备

3. **视频同步**
   - 选择视频文件
   - 自动转换为AVI格式（240x240, 5fps）
   - 上传到设备

## 使用方法

1. 运行应用
2. 点击"扫描设备"按钮，等待发现设备
3. 点击设备列表中的设备进行连接
4. 连接成功后，点击"同步图片"或"同步视频"按钮
5. 选择要同步的文件
6. 等待转换和上传完成

## 注意事项

- 需要授予蓝牙和存储权限
- 确保设备已开启蓝牙
- 视频转换可能需要一些时间，请耐心等待
